# PBJSCPRP_Hammer

Lua scripting and map modification tools for a Garry’s Mod SCP Roleplay server

Overview

This repository contains a collection of Lua scripts and map modification files developed for a custom SCP Roleplay (SCPRP) server in the game Garry’s Mod. The project focused on modifying existing maps, adding scripted functionality, and improving gameplay systems through custom Lua development.

The purpose of this repository is to showcase early game development work, including scripting, debugging, and working within an existing game engine framework all apart of a team of online developers.
