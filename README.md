# PBJSCPRP_Hammer
Hammer map editing and modification for a Garry’s Mod SCP Roleplay server

THIS PROJECT IS RETIRED

Overview

This repository contains map editing files and development resources created using the Hammer Editor for a custom SCP Roleplay (SCPRP) server in Garry’s Mod. The work focused on modifying and extending existing maps, creating new playable areas, and preparing map files for use on a live multiplayer server.

This repository is included as part of a portfolio to demonstrate early experience with level design, map editing, and working within the Source Engine, as well as collaborating as part of an online development team.
